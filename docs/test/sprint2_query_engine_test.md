# Sprint 2 功能测试文档 — 引擎层、在线查询、查询权限

> **测试版本：** SagittaDB v1.0-GA + v2-lite auth
> **对应计划：** Sprint 2 — 引擎层抽象、在线查询、查询权限申请
> **测试环境：** http://localhost（前端）/ http://localhost:8000（后端）
> **前置条件：** Sprint 1 测试通过，已创建至少一个可用数据库实例
> **测试账号：** admin / Admin@2024!（默认密码会触发强制改密；及已授权 query_query 权限的普通用户）

---

## 测试范围

| 模块 | 覆盖内容 |
|---|---|
| 引擎层 | 引擎注册表、多类型引擎切换 |
| 在线查询 | SQL 编辑器、查询执行、结果展示、行数限制 |
| SQL 安全控制 | DDL/DML 拦截、查询限制 |
| 数据库/表补全 | 自动补全实例下的库名、表名 |
| 查询权限 | 资源组实例范围、库级/表级授权、权限申请、审批、到期控制 |
| 禁用库拦截 | is_active=False 的数据库查询返回 403 |
| 数据库下拉框 | 禁用库管理员可见标灰"已禁用"、普通用户不可见 |

### TC-S2-015 PostgreSQL 非 public schema 表级权限

| 项目 | 内容 |
|---|---|
| **用例编号** | TC-S2-015 |
| **测试场景** | PostgreSQL 在线查询支持非 `public` schema 表级权限校验与执行 |

| **实际结果** | ✅ 通过（2026-04-20 本地与生产验证） |
