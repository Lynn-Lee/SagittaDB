# Pack B 功能测试文档 — 可观测中心

> **测试版本：** SagittaDB v1.0-beta
> **对应计划：** Pack B (Sprint 5+6) — Dashboard 统计、监控配置、Prometheus + Grafana 集成
> **测试环境：** http://localhost（前端）/ http://localhost:9090（Prometheus）/ http://localhost:3000（Grafana）
> **前置条件：** Pack A 测试通过，有若干历史工单数据
> **测试账号：** admin / Admin@2024!（默认密码会触发强制改密）；Grafana: admin / admin

---

## 测试范围

| 模块 | 覆盖内容 |
|---|---|
| Dashboard | 在线查询概览卡片、查询趋势、查询用户 Top 10、治理趋势、待审批库存趋势；SQL 工单概览卡片、趋势、排行图、按权限范围裁剪；实例与库概览卡片、实例类型/状态分布、库-Schema 状态分布 |
| 监控配置 | Prometheus Exporter 配置管理、告警规则 |
| Prometheus | 服务发现端点、指标采集 |
| Grafana | 面板加载、Dashboard 导入 |
| Alertmanager | 告警配置验证 |
| 会话诊断 | 在线会话、Kill 会话、平台历史会话、Oracle ASH 历史 |
| 慢日志分析 | 慢 SQL 明细、指纹聚合、指纹详情、采集配置、执行计划分析 |
