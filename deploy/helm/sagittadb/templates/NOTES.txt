SagittaDB 矢准数据 已部署！

访问地址：
{{- if .Values.ingress.enabled }}
  {{- range .Values.ingress.hosts }}
  - http{{ if $.Values.ingress.tls }}s{{ end }}://{{ .host }}
  {{- end }}
{{- else }}
  使用 kubectl port-forward 临时访问:
  kubectl port-forward svc/{{ include "sagittadb.fullname" . }}-frontend 8080:80
  然后访问 http://localhost:8080
{{- end }}

查看 Pod 状态：
  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }}

查看后端日志：
  kubectl logs -l app.kubernetes.io/component=backend -f

执行数据库迁移（迁移已在 initContainer 中自动运行）：
  kubectl exec -it deploy/{{ include "sagittadb.fullname" . }}-backend -- alembic upgrade head

  然后访问 http://localhost:5555
{{- end }}
